# Personal Portfolio Website

A clean, responsive personal portfolio website built for **Ravi Ranjan**, a 2nd-year B.Tech Computer Science student. This project showcases personal information, technical skills, and contact details with a modern design and theme-switching capabilities.

## 🚀 Features

* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Dark/Light Mode:** Includes a toggle button to switch between light and dark themes, saving the user's preference in local storage.
* **Profile Section:** Displays a profile picture, introduction, and bio.
* **Skills Showcase:** A visual list of technical skills (HTML, CSS, JS, React, etc.).
* **Contact Area:** Provides email and mobile contact information.

## 🛠️ Tech Stack

* **HTML:** Structure and semantic markup.
* **CSS:** Styling, variables for theming, CSS Grid, and Flexbox for layout.
* **JavaScript:** Logic for the theme toggle functionality.
